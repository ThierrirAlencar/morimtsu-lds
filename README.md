

## Description

Morimitsu-LDS is an complete api for morimitsu app. 

## Project setup

```bash
# Install dependencies
$ npm install --force

# Setup the database (With Docker)
$ docker compose up
```

## Compile and run the project

```bash
# development
$ npm run start



# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
